# Showcase Site Prompts — recording script

Account: **webdesigner@caddisfly.ai** (needs **agency tier** set in prod DB before
publishing — free tier caps at 1 published site). Curation afterward at
`/admin/showcase` from an ADMIN_EMAILS account (webdesigner does NOT need admin).

Per site: the wizard answers verbatim, plus which **features to tick in step 3**
and a **style** pick so the ten sites don't look alike. Mix up fonts/colors when
inspiration strikes — the AI takes the prompt from there.

## Recording-flow tips
- Vary the **language** on one or two (e.g. do the pizzeria-adjacent or salon in **Spanish/Portuguese**) — the generated site comes out fully localized, great demo moment.
- After generation: one **✨ AI edit**, one **photo swap**, one **theme switch** per video keeps each segment short but shows the editor.
- The 🔎 SEO panel arrives **pre-filled** (auto-SEO) — worth a 5-second callout.
- Publish each site (agency tier = unlimited) — the showcase tiles are live iframes, so sites must stay deployed.
- Curate at `/admin/showcase` with an admin account at the end; pick 3–4 as **featured** for the carousel.
